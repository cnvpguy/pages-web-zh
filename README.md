# pages-web-zh

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容无关特定域名或单一网站，仅作为静态网页文件的集合存储与展示。

## 项目简介

`pages-web-zh` 是一个面向中文内容的静态 HTML 页面归档仓库。仓库中收录的页面均为独立的 HTML 文件，可直接通过浏览器打开查看。本项目不绑定任何具体网站或域名，旨在以仓库形式集中管理和维护一批 HTML 资源。

## 目录说明

```
pages-web-zh/
├── README.md           # 本文件
├── index.html          # 仓库首页（可选）
├── pages/              # 存放独立 HTML 页面的目录
│   ├── example1.html
│   ├── example2.html
│   └── ...
└── assets/             # 公共资源（CSS、图片、脚本等）
    ├── style.css
    ├── images/
    └── scripts/
```

- `pages/`：核心目录，每个 HTML 文件为一个独立页面。
- `assets/`：存放页面共用的样式、图片或脚本文件，便于统一维护。

## 页面归档说明

- 所有页面均为纯 HTML 文件，不依赖服务器端处理。
- 页面之间相互独立，无数据互通或依赖关系。
- 归档内容可能包含各类静态页面，如说明页、工具页、展示页等。
- 文件命名保持清晰，避免使用特殊字符。

## 维护说明

- 新增页面：将 HTML 文件放入 `pages/` 目录，并确保其内部引用资源路径正确。
- 修改页面：直接编辑对应 HTML 文件，注意保持代码整洁。
- 删除页面：从 `pages/` 目录中移除文件即可。
- 公共资源更新：如需修改全局样式或脚本，请更新 `assets/` 下的对应文件。

## 使用方式

1. 克隆本仓库到本地。
2. 使用浏览器打开 `pages/` 目录下的任意 HTML 文件。
3. 或通过 GitHub Pages 等服务发布后在线访问。
